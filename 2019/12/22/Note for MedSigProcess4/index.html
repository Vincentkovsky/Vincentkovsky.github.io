ASCII- America Standard Code for Information Interchange

  1. 前32个字符(000(10)~031(10))，一般用于通讯或控制使用，比 如说换行。
  2. 接着的96个字符(032(10)~127(10))，用来表示阿拉伯数字，英文字母大小写和常用符号。

MATLAB RULE- String/numer

String 前后有单引号。 E.g. ‘1000’
Number直接输入 E.g. 1000
字符字串的转换：
abs(‘s’)- 字串转化为对应的十进制编码- ans=115
char(115)- 十进制编码转化为字符 - ans=s
abs(‘我很欣赏你’)

信号必要信息：

变量名称

  1. 变量名称由英文字母、数字或底线所组成
  2. 开头第一个字符是英文字母，不能是数字
  3. 名称长度不超过32个字符
  4. 会区分变量大小写
  5. 变量不用定义就可以直接用

注释快捷键

ctrl+R/ctrl+T
